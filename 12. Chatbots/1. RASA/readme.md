Step 1 : Install RASA

`pip install rasa`

Step 2: To check RASA is installed

`rasa -h`

Step 3: Initialise RASA Chatbot

`rasa init`

Step 4: To start chatting with the bot

`rasa shell`

Step 5: In case you make modification in the rasa training data and code

`rasa train`
`rasa test`
`rasa shell`

Step 6: To run Action Server

`rasa run actions`
